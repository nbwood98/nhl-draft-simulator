# nhl-draft-simulator
A TUI NHL draft lottery simulator built in Rust

## Installation

Download and install the latest release binary for your platform (Linux x86_64 or macOS ARM):

```sh
curl -fsSL https://raw.githubusercontent.com/nbwood98/nhl-draft-simulator/main/install.sh | bash
```

This downloads the binary to your current directory. Run it with:

```sh
./nhl-draft-simulator
```

Windows users can download the `.exe` manually from the [latest release](https://github.com/nbwood98/nhl-draft-simulator/releases/latest).
